Movie API

The movie web application is created from the server-side where users can access information about different movies, the cast, genre, and directors. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Allow new users to register /users POST A JSON object is holding data about the user to add: { "name": "Steve", "favoriteMovies": [], } A JSON object holding data about the user that has been added, which included an ID: { "name": "Steve", "favoriteMovies": [], "id": "3aa59c28-91cd-4393-ad21-4949c5760b5e" }
Allow users to update their user info /users/:id PUT A JSON object holding data about the user that needs to be updated: { "name": "Steve", "favoriteMovies": [] } A JSON object that holds data about the updated user information: { "id": 3, "name": "Steve", "favoriteMovies": [] }
Allow users to add a movie to their list of favorites /users/:id/:movieTitle POST None. User receives a message indicating whether the user added the movie successfully.
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None. User receives a message indicating whether the user removed the movie successfully.
Allow existing users to deregister /users/:id DELETE None User receives a message indicating whether the user deregistered successfully.
